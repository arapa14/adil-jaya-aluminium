# Storage Default Structure

Folder `storage_default` digunakan sebagai template struktur penyimpanan gambar pada project Laravel company profile.

Tujuan:
- Menjaga struktur folder tetap konsisten di repository
- Mempermudah onboarding developer
- Menjadi referensi lokasi asset image
- Menghindari upload asset production ke GitHub

## Catatan

Folder-folder di dalam `storage_default` hanya berisi file placeholder/default image
